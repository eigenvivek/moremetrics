# moremetrics

Add more image similarity metrics to TorchMetrics. 
Also wraps metrics from [`korina`](https://github.com/kornia/kornia).

Currently implemented:
- `ZeroNormalizedCrossCorrelation`
- `Carbonnier`
- `Welsch`
- `Cauchy`
- `GemanMcClure`

Requirements
- PyTorch
- TorchMetrics
- Kornia (latest version from github)
